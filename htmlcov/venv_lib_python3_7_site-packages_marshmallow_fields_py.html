65class Field(FieldABC): 

66 """Basic field from which other fields should extend. It applies no 

67 formatting by default, and should only be used in cases where 

68 data does not need to be formatted before being serialized or deserialized. 

69 On error, the name of the field will be returned. 

70 

71 :param default: If set, this value will be used during serialization if the input value 

72 is missing. If not set, the field will be excluded from the serialized output if the 

73 input value is missing. May be a value or a callable.

75 found in the input data. May be a value or a callable.
